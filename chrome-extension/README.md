## Prerequisites

- Node.js (>= 18.12.0)
- pnpm (package manager)

## Installation

1. **Install pnpm**:
    ```sh
    npm install -g pnpm
    ```

2. **Install dependencies**:
    ```sh
    pnpm install
    ```

## Development

### Start Development Server

To start the development server, run:

```sh
pnpm dev
```

This will start the server with hot module replacement enabled.

### Development for Firefox

To start the development server for Firefox, run:

```sh
pnpm dev:firefox
```

### Build

To build the extension, run:

```sh
pnpm build
```

For Firefox:

```sh
pnpm build:firefox
```

### Clean

To clean the build files, run:

```sh
pnpm clean
```

### Lint

To lint the code, run:

```sh
pnpm lint
```

To automatically fix lint errors, run:

```sh
pnpm lint:fix
```

### Prettier

To format the code with Prettier, run:

```sh
pnpm prettier
```

### Type Check

To check types, run:

```sh
pnpm type-check
```

### Test

To run tests, run:

```sh
pnpm test
```

## Load Extension in Chrome

1. Open Chrome and go to `chrome://extensions/`.
2. Enable "Developer mode" by toggling the switch in the top right.
3. Click "Load unpacked" and select the `dist` folder generated by the build process.
